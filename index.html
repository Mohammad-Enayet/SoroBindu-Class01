The main parts of our element are as follows: The opening tag: This consists of the name of the element (in this case,p), wrapped in opening and closing angle brackets. This states where the element begins or starts to take effect--in this case where the paragraph begins. The Closing tag; This is same as the opening tag, expect that it includes a forward slash before the elemennt name. This states where the elements ends--in this case where the paragraph ends. Failing to add a closing tag is one of the standard beginner errors and can lead tto strange results. The content: This is thhe content of the element, which in this caase, is just text. The element: The opening tag, and the content together comprise the element.
